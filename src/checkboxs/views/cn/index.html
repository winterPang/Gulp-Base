Checckbox复选框样式

绿洲OASIS提供一套基于绿洲的表单组件

ChecckBox 选中状态 选中状态 选中状态 未选中状态 未选中状态 
html:
<oasis-checkbox os-model="checkbox.values" value=1>选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox.values" value=2>选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox.values" value=3>选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox.values" value=4>未选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox.values" value=5>未选中状态</oasis-checkbox>
                             
js:
$scope.checkbox = {values:[1,2,3]}

os-model属性.设置双向绑定，通过在Scope作用域初始化绑定默认选中结果。
例如:
$scope.checkbox = {values:[1,2,3]}，chekBox设置的values数据必须是数组。通过点击选择我们可以修改其os-model的值。使其变为可以选中的状态。通过监听
$scope.$watch('checkbox1.values',(v,oldv) => {console.log(v); },true)处理checkBox选择事件。

禁用checkbox 禁用状态 选中状态 禁用状态 选中状态 禁用状态 
html:
<oasis-checkbox os-model="checkbox1.values" os-disabled="checkbox1.isDisabled" value=1>选中禁用状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox1.values" value=2>选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox1.values" os-disabled="checkbox1.isDisabled1" value=3>未选中禁用状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox1.values" value=4>选中状态</oasis-checkbox>
<oasis-checkbox os-model="checkbox1.values" os-disabled="checkbox1.isDisabled2" value=5>未选中禁用状态</oasis-checkbox>
                             
js:
$scope.checkbox1 = {values:[1,2,4]};

os-disable属性.设置双向绑定，通过在Scope作用域初始化绑定默认选中结果。
例如:
$scope.checkbox1.isDisabled=true;。使其禁用状态。通过监听
$scope.$watch('checkbox1.isDisabled1',(v,oldv) => {console.log(v);})处理checkBox状态变化事件。

属性配置
元素 属性 类型 可选值 默认值 说明
oasis-checkbox os-model string或number或boolean ------- ------- os-model属性。指令实现双向绑定。可以通过初始化设置其值与value值一致实现默认选项。也可以动态赋值，改变其选择选项。通过监听其值的变化，处理用户选择事件
os-disabled boolean ------- ------- os-disabled指令实现双向绑定,通过设置其boolean类型的数值，触发单选按钮的禁止。通过监听其值可以监听单选按钮的状态变化
value string或number或boolean ------- ------- value属性类似于radio元素的vaule属性，预设改组的radio的值
