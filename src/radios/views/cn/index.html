Radio单选按钮样式

绿洲OASIS提供一套基于绿洲的表单组件

radio 选中状态 未选中状态

 
html:
<oasis-radio os-model="radio.isChecked" value="checked">选中状态</oasis-radio>
<oasis-radio os-model="radio.isChecked" value="unchecked">选中状态</oasis-radio>
js:
 $scope.radio={isChecked:'checked'}
$scope.$watch('radio.isChecked',function(v){console.log(v)})

os-model属性.设置双向绑定，通过在Scope作用域初始化绑定默认选中结果。
例如:
$scope.radio1 = { isChecked:'checked' }，通过点击选择我们可以修改其os-model的值。使其变为可以选中的状态。通过监听
$scope.$watch('radio1.isChecked'function(v){})处理radio选择事件。

禁用radio按钮 选中状态 未选中状态 
html:
<oasis-radio os-model="radio.isChecked" os-disabled="radio1.isDisable" value="checked">选中状态</oasis-radio>
<oasis-radio os-model="radio.isChecked" os-disabled="radio1.isUncheckedDisable" value="unchecked">选中状态</oasis-radio>
js:
 $scope.radio1={isDisable:true}
$scope.$watch('radio1.isDisable',function(v){console.log(v)})

os-disabled属性.设置双向绑定，通过在Scope作用域初始化绑定默认是否需要禁用radio按钮。
例如:
$scope.radio1 = { isDisable:true }异步使其失效，或者监听改值处理当被禁用时候的事件处理

属性配置
元素 属性 类型 可选值 默认值 说明
oasis-radio os-model string或number或boolean ------- ------- os-model属性。指令实现双向绑定。可以通过初始化设置其值与value值一致实现默认选项。通过监听其值的变化，处理用户选择不同选项事件
os-disabled boolean ------- ------- os-disabled指令实现双向绑定,通过设置其boolean类型的数值，触发单选按钮的禁止。通过监听其值可以监听单选按钮的状态变化
value string或number或boolean ------- ------- value属性类似于radio元素的vaule属性，预设改组的radio的值
